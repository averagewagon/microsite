Joni Hendrickson's Microsite

Microsite is a blog hosted entirely on an ESP32-S3 microcontroller, demonstrating how modern web design can thrive even on severely resource-constrained hardware. This project explores how much functionality can be squeezed out of a small microcontroller, using innovative techniques to serve web content efficiently and reliably.

Whether you're a web developer, embedded systems enthusiast, or just curious about pushing hardware to its limits, Microsite offers a hands-on look at what’s possible when you strip web hosting down to the bare essentials.

About the Project

The Microsite project was born out of a desire to explore the intersection of web design and embedded systems. By hosting a blog on an ESP32-S3, I’ve combined my love of microcontrollers with the principles of modern, fast-loading websites.

This project includes:

  • Custom static site generator to minimize dependencies.
  • Base64-encoded assets for full self-containment.
  • Efficient, small-scale hosting powered entirely by a microcontroller.

Key Features

Hardware

The site runs on an ESP32-S3, chosen for its balance of processing power, energy efficiency, and onboard features. The hardware setup includes:

  • A lightweight HTTP server optimized for embedded devices.
  • Secure HTTPS connections using LetsEncrypt certificates.
  • Support for multi-page hosting and serving static files.

Optimizations

To ensure smooth operation on limited hardware, the Microsite employs various optimization strategies:

  • Single `.html` file delivery for each page, reducing resource load.
  • Minified CSS, JavaScript, and other assets.
  • Base64-encoded images to eliminate external file dependencies.
  • Brotli-compressed payloads and ultra-compressed webp images for fast transfer and reduced storage requirements.

Future Developments

The Microsite is a continuously evolving project. Planned future developments include:

  • Setting up personal image and file storage on a microcontroller for lightweight hosting needs.
  • Experimenting with hosting a git server entirely on microcontroller hardware.
  • Developing a microcontroller-based load balancer to distribute traffic across multiple devices, creating a fleet of microcontroller-powered web servers.

Live Demo

The Microsite is live at joni-on-micro.site. You can explore the site in real time, experiencing the optimizations and constraints of hosting on a microcontroller.
